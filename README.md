# Matriz caseira

O código apenas mostra os valores de uma matriz boleana nos leds.

conexão:

```
    
(positivo)   pin 8      pin 9      pin10
(negativo)    |          |          |
              |          |          |
pin 5 ------ led ------ led ------ led
              |          |          |
pin 6 ------ led ------ led ------ led
              |          |          |
pin 7 ------ led ------ led ------ led  

```
